| Key | Kind | Type | Default Value | Description |

| <span id="a6a064-content-plain"></span> `content-plain` | `VALUE` | `String` |   | Plain content of the resource (text) |
| <span id="af0a15-description"></span> `description` | `VALUE` | `String` |   | Description of this resource when configured through plain text or binary |
| <span id="af326f-path"></span> `path` | `VALUE` | `Path` |   | Resource is located on filesystem |
| <span id="a4a472-proxy-host"></span> `proxy-host` | `VALUE` | `String` |   | Host of the proxy when using URI |
| <span id="ab6b48-proxy-port"></span> `proxy-port` | `VALUE` | `Integer` | `80` | Port of the proxy when using URI |
| <span id="a0bf08-resource-path"></span> `resource-path` | `VALUE` | `String` |   | Resource is located on classpath |
| <span id="a4ef3f-uri"></span> `uri` | `VALUE` | `URI` |   | Resource is available on a `java.net.URI` |
| <span id="aa585f-use-proxy"></span> `use-proxy` | `VALUE` | `Boolean` | `true` | Whether to use proxy |
