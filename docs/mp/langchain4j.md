# LangChain4j

## LangChain4j Support

Helidon SE includes native [LangChain4j integration][langchain4j-inte], which
can also be used in Helidon MP thanks to **Helidon Inject to CDI bridge**. Refer
to its [documentation][langchain4j-inte] for more details.
